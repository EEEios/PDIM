## 自定义协议格式

| 表头   | 版本号 | 序列化方法 | 指令类型 | 长度   | 内容 |
| ------ | ------ | ---------- | -------- | ------ | ---- |
| 4 byte | 1 byte | 1 byte     | 1 byte   | 4 byte |      |



## 指令

| 指令                      | 功能     |
| ------------------------- | -------- |
| Logout                    | 退出登录 |
| send <username> <message> | 发送消息 |
| eng <groupname>           | 加入群聊 |



## 创建新命令流程

1. Commend 添加指令
2. 创建对应包
3. 在 PacketCodec 中加入解析
4. 创建命令处理器
5. 创建请求/响应处理器
